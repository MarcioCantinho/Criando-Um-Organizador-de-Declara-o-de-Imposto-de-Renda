# Criando-Um-Organizador-de-Declara-o-de-Imposto-de-Renda
Solução completamente construída no Excel, este projeto teve como objetivo criar uma ferramenta para organizar e reunir informações essenciais para a declaração de imposto de renda.
A proposta foi construir um agregador de dados onde o usuário pudesse controlar suas entradas de maneira eficiente e validada, com menus de navegação, validações automáticas e funcionalidades extras. 
Obs: Algumas funcionalidades não puderam ser feitas de acordo com a aula pois a aplicação foi feita na versão online do programa, com limitações de uso.
